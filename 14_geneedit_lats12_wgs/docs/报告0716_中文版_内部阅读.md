## 1. 研究目的

**Study A — 编辑细胞 → 体内成瘤(Trp53⁺/⁻;Cas9 背景)**

- **A1** 验证预期的 CRISPR 编辑(Brca1、Brca2、Pten 敲除)是否真实发生。
- **A2** 表征所成肿瘤的基因组(体细胞改变、copy-number/aneuploidy),并判定每个瘤的来源谱系(B1TP = Brca1+Pten vs B2TP = Brca2+Pten)。

**Study B — Lats1/2-flox 组织,± iHPV,年龄序列(3M/12M/18M)**

- 判断这些组织的基因组相对正常 C57BL/6 是否存在异常(aneuploidy、structural/de-novo variants),以及是否可能解释客户报告的 >10 月龄输卵管表型。
- 定位 iHPV transgene,评估其整合是否打断候选基因。

**关键方法学前提**:Study B 的工程元件按设计**处于 loxP-only / Cre 依赖的未激活状态**(Lats1/2 已 flox 但未删除;E6/E7 位于 lox-stop-lox 之后、未表达)。因此分析围绕 insertional / somatic / de-novo 假设展开,而不是假定 "Lats1/2 缺失 → 不稳定" 这条机制。

## 2. 核心结论(Key Findings)

1. **12 个样本全部通过 QC** — 99.99% 读段比对到 GRCm39,duplicate 率 7.7–14.3%,平均深度 19.2–31.9×,84–92% 基因组 ≥10×。无样本被剔除,下述结论均不受数据质量限制(§6.1)。
2. **所有预期 CRISPR 编辑均已验证,肿瘤谱系已判定。** B1TP 确认 Brca1 + Pten 敲除。**B2TP 的 Brca2 敲除确认,且为双等位(biallelic)** —— 无任何野生型 Brca2 等位存留 —— 同时带 Pten 敲除。(敲除本身证据扎实;但**每条等位上具体是什么病灶**在此深度无法解析,报告未作此声称 —— §6.3。)tumor1 与 tumor2 来自 **B1TP(Brca1+Pten)** 谱系。
3. **tumor3 不带任何预期编辑,来自亲本的未编辑亚克隆。** 它在三个靶基因(Brca1、Brca2、Pten)全为野生型,但 SNP-fingerprint 显示它与亲本细胞系**是同一只小鼠** —— 故为 editing-escaped 亚克隆,而非样本混淆。**实践后果:编辑逃逸的细胞在本模型中仍能成瘤**(tumor3 恰恰是三个瘤里最非整倍体的),因此不应在未对该具体肿瘤做基因分型的情况下,把肿瘤表型归因于 Brca/Pten 缺失。
4. **三个肿瘤全部丢失 Trp53 —— 这是驱动它们的 second hit。** 亲本为 Trp53⁺/⁻;每个瘤都在 Trp53 变成纯合,而编辑后未成瘤的细胞没有。tumor1 的纯合区是 **focal、精确套在 Trp53 上**(69–70 Mb 处 92%,而两侧仅 3 Mb 外为 1–3%)—— 这是**选择性丢失 Trp53 的直接证据**(§6.5)。
5. **tumor3 经历了全基因组灾难:95% 的基因组失去杂合性,且拷贝数仍为二倍体。** 这是 genome-wide copy-neutral LOH —— 丢失一整套单倍体基因组后另一套加倍。它使 tumor3 在**所有**抑癌基因上同时纯合,并解释了"唯一不带任何工程编辑的瘤为何最非整倍体"(§6.6)。
6. **Study A 三个肿瘤均为非整倍体,各自核型不同。** 染色体级 gain/loss 各异,且**没有任何一条染色体在两个瘤中朝同方向改变** —— 独立的克隆演化。亲本与编辑后未成瘤细胞仍为二倍体。
7. **Study B 组织基因组稳定 —— 无 aneuploidy。** 六个 Lats1/2-flox 组织(两种基因型、所有年龄)全基因组 copy-number 平坦二倍体(常染色体 CN 1.94–2.08)。输卵管表型**不是**大尺度染色体不稳定或非整倍体驱动的。**但这只排除了一个机制,不是全部**:设计阶段列为最可能的两个假设 —— iHPV insertional mutagenesis(整合位点未解析)与 leaky loxP recombination(判据尚未有效)—— **仍未测试**,故这**不是**"未发现基因组原因"的结论。状态详见 §6.10。
8. **表型影响两个品系,但只有一个带 transgene** —— 故 iHPV 整合不可能是完整解释。两品系共有的是 floxed Lats1/2 等位与非 C57BL/6J 背景;两者都是活跃候选,而前者(hypomorphic floxed allele)**WGS 看不见**,需 RNA 或蛋白检测(§6.10)。
9. **iHPV transgene 特异性存在于 L1L2H。** HPV16 E6/E7 与 luciferase 读段在三个 L1L2H 样本全部检出,在三个 L1L2 样本**完全不存在**(完美特异性),丰度与固定的 germline transgene 一致。
10. **Study B 背景不是同源纯 C57BL/6J。** 每个组织相对 GRCm39 带 ~5–6 百万变异 —— 全基因组范围的非-6J 近交系背景,最可能是 ES 细胞工程带来的 129 型。这是**遗传性背景**而非累积突变,并决定了 de-novo 候选必须如何挖掘(§6.9)。

## 4. 分析依据与判定标准

| 问题 | 方法,及为何无需专门野生型对照 | 阈值 / 判据 |
| :--- | :--- | :--- |
| Copy number / aneuploidy | 基因组内部 coverage 比值 → copy number;参考-free。GRCm39 = C57BL/6J ≈ 这些动物自己的正常。 | 染色体中位 CN ≥2.5 = gain,≤1.5 = loss |
| 编辑验证 (A1) | 直接读每个 sgRNA 切点;编辑/肿瘤样本 vs RO_origin 亲本。**落到 read/CIGAR 层面,不信 caller 输出** — 见 §6.3 | 无野生型读段 ⇒ 双等位敲除;编辑样本有 indel、RO_origin 为 0/0 |
| soft-clip 能否作证据 | 本位点固有易 clip(连未编辑亲本都有 ~30% 读段被 clip),故 clip **比例**本身不特异 | 仅当 clip ≥20 bp **且**断点**反复堆叠在同一切点碱基**上才计入 |
| 阴性对照闸门 | 未编辑亲本必须判为未编辑,否则方法有问题 | RO_origin 必须 ~0% edited(已在代码中硬断言) |
| 肿瘤谱系 (A2) | Brca1-KO ⇒ B1TP 谱系;Brca2-KO ⇒ B2TP 谱系。**无野生型等位的细胞系不可能产生带野生型的后代** | 切点 indel 有无;野生型读段比例 |
| 样本身份 (A2) | 朴素 genotype concordance 会被肿瘤 LOH(het→hom)混淆,系统性冤枉最非整倍体的样本;**private allele** 对此免疫 —— LOH 只移除等位,不可能造出亲本没有的等位 | 纯合 private allele ≈ 0 ⇒ 同一只鼠;成千上万 ⇒ 不同个体 |
| Trp53 second hit | coverage 只看得见 deletion 型等位;**LOH** 无需知道等位设计即可判定 | origin 杂合的 SNP 在后代是否变纯合;与该样本自身全基因组基线比 |
| iHPV 存在 / 整合 (B) | Transgene 读段在纯 GRCm39 下为 unmapped → 提取后比对 HPV16/EGFP/Luc 标记;L1L2 = 内部阴性对照 | 构建体读段在 L1L2H 有、L1L2 无 |
| De-novo 候选 (B) | 从各样本 germline calls 中扣除已知小鼠品系背景(Sanger MGP,全品系) | 样本私有(不在 MGP)、且复现或基因型差异 |

## 5. 方法

- **比对 / 预处理**:nf-core/sarek 3.8.1,bwa-mem2,GRCm39(GENCODE vM35),标记重复,跳过 BQSR。
- **变异检出**:Study A somatic = GATK Mutect2(每个 tumor vs RO_origin)+ TIDDIT SV;Study B germline = GATK HaplotypeCaller + CNNScoreVariants + TIDDIT SV。
- **Copy number**:mosdepth 500 kb 分箱深度 → 按样本常染色体中位归一 → copy number = 2 × ratio;染色体级中位 + 队列热图(小鼠染色体为端着丝粒,故全染色体 CN ≈ 臂级)。
- **Guide 定位**:每条 spacer 按序列在 GRCm39 双链搜索(`seqkit locate`);PAM 由 protospacer 3' 紧邻 3 bp 确认;切点 = PAM 上游 3 bp。
- **编辑验证**:切点窗口多样本 `bcftools mpileup`/`call`(MAPQ≥20, BAQ≥15),**外加 read 层面等位定量**(自建 pysam 分析):MAPQ≥20、剔除 duplicate,按 CIGAR 分类为野生型(跨核心区且两侧 ≥25 bp margin、无 indel/clip)、切除等位、其它切点 indel、切点 soft-clip(≥20 bp 且断点在核心区内);并按碱基统计 clip 断点以区分**反复堆叠**与背景。
- **样本身份指纹**:六个 Study A 样本在 38 个 × 500 kb 窗口(共 19.0 Mb,覆盖全部 19 条常染色体)联合 `bcftools mpileup/call`,保留双等位 SNP。"private allele" = 该样本带 ALT 且 ≥3 条支持读段,而 RO_origin 在 ≥10× 深度下 **0 条** ALT。结果按染色体与等位分数分别拆解。
- **LOH 分析**:取 RO_origin 为 0/1(杂合、DP≥10)的位点,统计各后代是否变为 1/1 或 0/0;并沿 chr11 扫描以定位 tract 边界;VAF 谱作为独立于 genotype calling 的硬证据。
- **iHPV 整合**:按样本提取 unmapped 读段,比对(bwa-mem2)至标记参考(HPV16 NC_001526.4 + EGFP + firefly luciferase);由 mate 未比对的已比对读段提取基因组锚点。
- **De-novo 候选**:用 Sanger Mouse Genomes Project v8(REL-2021, GRCm39)SNP+indel VCF,以 `bcftools isec` 从各样本 germline calls 中扣除(染色体名 chr1↔1 已统一)。

- **比对基本完全** —— 每个样本 99.99% 的 QC 通过读段比对上 GRCm39(如 RO_origin:458.9 M 中仅 53,411 条 unmapped)。上游 fastp 保留 99.0% 原始读段,故约 99% 的全部测序读段被用上。

### 6.3 CRISPR 编辑验证(Study A)

**一个决定结果的方法学要点。** 联合 `bcftools` call 在**任何样本的任何 Brca2 切点都没报出 indel**。若照单全收,读起来就是"Brca2 编辑没发生" —— 而这是**错的**。B2TP 切点上 25 条读段里只有 **1 条**是完美匹配(`150M`),而未编辑亲本是 34 条里 24 条:编辑等位确实存在,只是带 31 bp 缺失或被大幅 soft-clip(50–87 bp),caller 在这种复杂位点没能解析。故下述 Brca2 结论建立在 **read/CIGAR 层面**的证据上。

- **B1TP**:Brca1 KO + Pten KO —— 与其 Brca1+Pten 设计相符。
- **B2TP —— Brca2 敲除确认,双等位。** 两条独立证据确立:
  - **无野生型 Brca2 等位存留。** B2TP 是唯一**野生型 fragment 为 0** 的样本(0/9),而其余样本都保有明确的野生型群体(4–11 个)。若还有一条 Brca2 是野生型,这 9 个 fragment 中约一半应为野生型;一个都没观察到的概率约为 **0.002**。
  - **覆盖只在切点塌陷,别处不塌。** B2TP 的 Brca2 全基因覆盖正常(23.2×),却在切点窗口降到**自身基线的 0.57**(13.3×)—— 正是编辑分子无法干净比对的预期后果;其余样本均在 0.87–1.23。**该度量完全不依赖 read 计数。**
- **两个等位各自如何被破坏,尚未解析;数据不支持给它们命名。** 切点窗口有效深度仅 ~7×,合计 9 个有效 fragment —— 足以确立"无野生型等位存留",但**不足以重建两个等位的结构**。具体:
  - **预期的切除被观察到,但只在 1 个分子上。** 1 个 fragment 带 chr5:150,452,958–150,452,988 的 31 bp 缺失,删除 `TGGTTTGAGGAGCTTTCCTCAGAAGCCCCCC`。其两端不只是靠近 guide —— 5' 端紧邻重叠 guide 对的切点(150,452,957/961)、3' 端紧邻第三条 guide 的切点(150,452,989),正是三条 guide 同时切割时预期被切除的片段。它完全位于 CDS exon 3 内、起自 CDS 第 91 位核苷酸(**约第 31 个密码子 / 全长 3,329**),且不是 3 的倍数,故会造成 frameshift 与蛋白前 1% 内的提前终止。**但仅 1 个分子,这是一个相符的观察,不是已确立的等位** —— 应读作"该切除在这株细胞系中发生过",而非一个基因型。
  - **多数被破坏的分子断在第三条 guide 的切点上。** 9 个 fragment 中有 7 个被 soft-clip,其断点**堆叠在单一碱基**(150,452,989/990)上,而非像其余样本那样零散分布。被 clip 的序列在别处只能模糊比对(MAPQ 0,重复序列),故它究竟是复杂 indel、插入还是更大的重排,短读长无法解析。
  - 另有 1 个 fragment 在第三个切点带 3 bp 缺失;**1 个分子不构成等位证据**,此处不作解读。
- 因此 **B2TP = Brca2 + Pten 敲除** 是由**直接观察到 Brca2 两条等位均被破坏**确立的,而非从其 Pten/Brca1 模式推断。要弄清每条等位上的确切病灶,需要针对切点的**定向深测序**(amplicon 或长读长),而不是更多 WGS。
- **tumor1、tumor2**:同时带 Brca1 与 Pten indel → **B1TP(Brca1+Pten)谱系**;两者的 Brca2 均为野生型,符合预期。
- **tumor3**:**三个**靶基因全为野生型。其唯一的 Brca2 indel call —— 150,452,983 处、仅 1 个 fragment 上的 1 bp 缺失 —— 落在一段 **7-C homopolymer** 内(`AAG`**`CCCCCCC`**`ATACAATTCTG`,150,452,983–150,452,989),这是典型的测序假象场景,已不予采信。谱系逻辑是决定性的:**B2TP 没有野生型 Brca2 等位**,故 B2TP 的任何后代都不可能重新获得一个;tumor3 在该位点约 47% 为野生型,因此**不可能来自 B2TP**。Pten 从另一方向同样决定性 —— **两条**注射细胞系都带 Pten 编辑(B2TP 为纯合),故来自任一条都会留下可检出的 Pten 病灶,而 tumor3 一个也没有(Pten 位点 17–18 条读段,明确为参考 —— 是真的没有编辑,不是覆盖空洞)。tumor3 不来自任何一条编辑细胞系。

### 6.4 tumor3 身份 —— 与亲本是同一只小鼠

tumor3 在三个靶基因全为野生型,这留下一个**身份**问题:它是亲本的未编辑逃逸亚克隆,还是来自另一只小鼠的样本混淆?B1TP/B2TP/tumor1/tumor2 —— 已知都源自 origin —— 提供"同鼠"基线。共 19,032 个可评估位点。

**设计。** 朴素的 genotype concordance 在此会误导:肿瘤带大量 copy-number 缺失与 LOH,会把杂合位点转成纯合、压低一致性 —— 而 tumor3 恰恰是三个瘤里**最**非整倍体的,朴素判据会系统性地冤枉正被怀疑的那个样本。改用 **private allele**:该样本带某等位,而 RO_origin 在同位点(≥10×)**0 条**支持读段。LOH 只能**移除**等位,不可能造出亲本从未有过的等位。38 个窗口共 19.0 Mb,覆盖全部 19 条常染色体。

| 样本 | private allele(原始) | 剔除 chr3/chr19 | **纯合 private(剔除后)** |
| :--- | :---: | :---: | :---: |
| RO_B1TP | 245 | 223 | 0 |
| RO_B2TP | 227 | 216 | 0 |
| RO_tumor1 | 264 | 231 | 0 |
| RO_tumor2 | 219 | 182 | 1 |
| **RO_tumor3** | **2,851** | 644 | **8** |

**原始计数是个陷阱,列出来是因为它有教育意义。** 表面上 tumor3 的 2,851 个 private allele(约为其余样本的 11 倍)看着就是另一只鼠。两次下钻证明不是:

- **按染色体拆**:tumor3 的 1,218 个纯合 private allele 中,**1,154 个挤在 chr3 的单个 500 kb 窗内(~47.9–48.4 Mb)**,56 个在 chr19;**其余 17 条常染色体贡献为零**。真正不同的个体会在**每一条**染色体上都有差异。这是局部事件 —— 而且该 chr3 窗口在 tumor3 中覆盖异常(仅为自身基线的 0.35×,其余样本 0.69–1.00),即一个肿瘤特异的结构性丢失,其残留的错配读段制造了假的纯合 call。
- **按等位分数拆**:剔除这两个区域后,tumor3 只剩 **8 个纯合 private allele**。其余多出来的 private 集中在 **VAF 0.3–0.7** —— 这是**克隆性体细胞突变**的特征(它是三个瘤里重排最重的),而非遗传性变异(后者会堆在 VAF ~1.0)。

**结论:** tumor3 与 RO_origin 共享 germline 基因组。是同一只鼠,与另一只小鼠的样本混淆已被排除。结合 §6.3,**tumor3 来自 Trp53⁺/⁻;Cas9 亲本的未编辑(编辑逃逸)亚克隆**。

**一个必须明说的局限:** 本方法区分的是*不同的基因组*,而非*近交系中的不同个体*。近交系同窝个体基因组近乎相同,故这类混淆对任何指纹方法都是隐形的。已确立的是:tumor3 的基因组就是亲本的基因组 —— 这正是"未编辑亚克隆"解释所要求的,也是"与无关或不同工程背景的动物混淆"会破坏的。

### 6.5 杂合性丢失 —— 三个肿瘤全部丢失 Trp53,机制各不相同

亲本按设计是 **Trp53⁺/⁻**,故走向 Trp53-null 肿瘤的经典路线是通过 "second hit" 去掉剩下那条功能性等位。**coverage 看不见这件事**:六个样本(含亲本)的 Trp53 读段深度都平坦不变,说明那个工程 null 等位是小病灶而非缺失。**LOH 能回答,而且完全不需要知道等位设计** —— 只需问:在 RO_origin 中杂合的 SNP,在各后代中是否变成了纯合。B1TP 与 B2TP —— 已编辑但从未经过成瘤 —— 是阴性对照,定义噪声底。

**全基因组 LOH(19 条常染色体上 11,272 个 origin 杂合 SNP):**

| 样本 | 全基因组 LOH |
| :--- | :---: |
| RO_B1TP | 5% |
| RO_B2TP | 4% |
| RO_tumor1 | 10% |
| RO_tumor2 | 13% |
| **RO_tumor3** | **95%** |

**沿 chr11 扫描 Trp53 位点(Trp53 = 69.47 Mb):**

| chr11 窗口 | 有效 SNP | B1TP | B2TP | **tumor1** | tumor2 | tumor3 |
| :--- | :---: | :---: | :---: | :---: | :---: | :---: |
| 55–56 Mb | 367 | 3% | 3% | **3%** | 94% | 95% |

| **69–70 Mb(Trp53)** | 725 | 2% | 2% | **92%** | 93% | 94% |

**三个肿瘤在 Trp53 全部纯合;三个未成瘤细胞都不是。** 机制各不相同,而这恰恰是重点 —— 三条独立路径汇聚到同一个位点:

- **tumor1 —— 一段以 Trp53 为中心的 focal LOH。** 这是决定性观察。tumor1 的全基因组 LOH 仅 10%,在 chr11 上 **65–66 Mb 处杂合(1%)、72–73 Mb 处杂合(3%)**,却在 **69–70 Mb(Trp53 所在)高达 92%**。两侧仅 3 Mb 外杂合性完好。随机事件不会在一个抑癌基因上精确刻出一段窄纯合区然后停住 —— 这是**选择性丢失 Trp53**,直接写在数据里。
- **tumor2 —— chr11 大片段区域性 LOH**(55 Mb 至 73 Mb 处 94–98%)包含 Trp53,而其全基因组基线为 13%。
- **tumor3 —— 全基因组 LOH**(§6.6);其 Trp53 纯合是整个基因组一起来的,而非局部靶向。

**这确立了什么、没确立什么。** 每个瘤只保留一条 Trp53 单体型,故均为纯合 —— 要么 null/null,要么野生型/野生型。要区分保留的是哪一条,需要那个工程 "−" 等位的设计(病灶到底是什么),我们没有;有了它,现有数据足以直接分型。但这个推断并非势均力敌:保留野生型那条会**恢复** p53 功能、不带来任何优势,而丢掉它正是 Trp53⁺/⁻ 肿瘤的经典 second hit。三个独立肿瘤汇聚到 Trp53 纯合(其中一个还是 focal),正是**选择丢失 Trp53** 的样子,并且提供了 Brca/Pten 编辑在 tumor3 中无法解释的那个驱动因素(§7.1)。

### 6.6 tumor3 经历了全基因组 copy-neutral LOH

tumor3 的 LOH 不局限于 Trp53、也不局限于任何一条染色体:**全基因组 95% 的 origin 杂合位点在 tumor3 中已变纯合**,而编辑后未成瘤细胞仅 4–5%。等位分数谱把这一点坐实到任何 genotype-calling 假象都无法解释的程度 —— 在亲本为干净杂合的位点上,tumor3 的读段**双峰分布于 0 与 1**,0.5 附近几乎为零:

| ALT 等位分数 | RO_origin | RO_B1TP | RO_B2TP | RO_tumor1 | RO_tumor2 | **RO_tumor3** |
| :--- | :---: | :---: | :---: | :---: | :---: | :---: |
| 0.0–0.1 | 2.0% | 5.1% | 4.6% | 9.2% | 9.6% | **40.7%** |
| 0.4–0.6(杂合) | 50.7% | 49.1% | 52.2% | 19.3% | 38.1% | **1.0%** |
| 0.9–1.0 | 0.0% | 0.2% | 0.1% | 1.6% | 2.6% | **53.2%** |
| 评估位点数 | 3,433 | 1,711 | 1,865 | 2,430 | 1,010 | 2,528 |

tumor3 基本上已无杂合位点。关键在于,**其拷贝数在大部分基因组上仍是二倍体** —— chr2 2.06、chr7 2.03、chr16 2.09、chr18 2.19 —— 而这些同样的染色体有 94–99% 已纯合。**正常拷贝数下的纯合 = copy-neutral LOH**,而全基因组的 copy-neutral LOH 只有一个标准解释:**丢失一整套单倍体基因组,随后另一套加倍**(haploidisation + endoreduplication)。§6.2 中的那些 aneuploidy(chr8、chr12、chr13、chr14 缺失;chr15 增益)是叠加在该事件之上的。

这是本队列中最极端的基因组表型,而它属于**唯一不带任何预期编辑的那个瘤**(§6.3)。两点值得明说:

- 它一次性让 tumor3 在**每一个**抑癌基因上纯合,Trp53 也在内 —— 一条通往 Brca/Pten 编辑本欲工程化实现的同一终点的、灾难性的单一路径。
- 它印证了 §6.4 所用的身份判据。朴素的 genotype concordance 会把 tumor3 判为约 95% 不一致,从而"证明"样本调包。选用 private-allele 判据正是因为 LOH 移除等位却不能创造等位 —— 而 tumor3 恰恰是那个会击穿任何 concordance 类检验的极端案例。

**这些是每样本的原始 call,不是 somatic SV,绝对数不能当作 SV 负荷读。** 它们是相对参考的计数,因而包含该样本携带的每一个遗传性 SV。Study B 的计数(9,000–14,500 PASS)是 Study A 的两到三倍,原因正在于此 —— 那些组织坐落在非-6J 品系背景上(§6.9),它们的多数"SV"是相对 GRCm39 的遗传性品系差异,而非病灶。只有在**同一臂内、背景匹配**的前提下比较才有意义。

**在 Study A 内部,有一个信号越过了这个 caveat:tumor3 是明显离群** —— 5,416 个 PASS,而亲本、两条编辑细胞系与另外两个瘤为 3,362–3,880(约为最高者的 1.4 倍、最低者的 1.6 倍),主要由 deletion 驱动(4,347 vs 2,694–3,114)。六个 Study A 样本共享同一遗传背景,故这个超出不是背景假象。它独立佐证了 copy-number 结果(§6.2):**tumor3 是三个瘤里基因组重排最重的** —— 而它正是那个不带任何预期编辑的瘤(§6.3)。

两点判读限制:此处 TIDDIT 计数是每样本的,而非 tumor-minus-normal(相对 RO_origin 的 somatic SV calling 列为后续项,§7.3);且短 insert size(§6.1)在所有样本上一致地降低了 paired-end SV 灵敏度。两者都不影响 Study A 内部的相对比较。

### 6.9 De-novo 候选变异(Study B)

Sanger Mouse Genomes Project(MGP)是一个记录常见实验小鼠品系(如 129、C57BL/6 亚系、BALB/c)相对 GRCm39 参考之 DNA 变异的目录;扣除它即移除遗传性品系背景,只留下真正为样本私有的变异。各样本的 germline calls 据此对 Sanger MGP v8(全品系)SNP+indel 目录作了扣除:

| 样本 | 相对 GRCm39 的 germline 变异总数 | MGP 扣除后 private |
| :--- | :---: | :---: |
| L1L2_3M | 5,101,717 | 573,954 |
| L1L2H_3M | 5,334,890 | 562,308 |
| L1L2_12M | 6,110,749 | 658,142 |
| L1L2H_12M | 5,612,998 | 589,237 |
| L1L2_18M | 5,899,863 | 625,093 |
| L1L2H_18M | 5,884,909 | 615,276 |

**判读 —— 一个重要 caveat。** 两点观察表明这些计数**不是**因果性 de-novo 候选清单:

- 每个样本携带**相对 GRCm39 约 5–6 百万变异** —— 这是个确凿且有信息量的结果。由于 **GRCm39 就是 C57BL/6J("6J")参考组装**,一只真正纯的 C57BL/6J 动物与它只差数万个变异(常规群体漂变)。数以**百万**计因此是清晰信号而非噪声:**这些组织携带全基因组范围的非-6J 经典近交系背景 —— 即该品系并非同源纯 C57BL/6J。** 这是遗传性品系背景,**不是**该品系建立以来累积的突变:多世代繁育、培养时间、以及基因编辑本身,三者合计最多只增加数万个*新的、样本私有的*变异(比 5–6 M 少两到三个数量级);而同品系漂变的经验上限 —— C57BL/6J 与 C57BL/6N 分家约 70 年 —— 同样只有数万。决定性的是,**此处约 90% 的负荷(5–6 M 中的约 4.5–5.4 M)能在已知 MGP 品系中查到**并被扣除;一个变异能进 MGP,当且仅当它是遗传性品系多态,而培养/繁育/编辑产生的突变是样本私有的、**不会**被扣掉。
- **最可能的来源(educated guess):129 来源的工程背景。** 基因打靶与 floxed 等位通常在 **129 来源的胚胎干细胞**中制作,除非完全回交至 C57BL/6J 同源纯合,否则 129 背景会全基因组携带。本品系的 ES 细胞来源在所提供材料中无记载,故我们将 129 作为主导解释而非定论 —— 但它证据充分:129 既是此类等位的标准 ES 细胞供体,也在数量上吻合,因为 **129 品系与 6J 参考相距约 4–6 百万 SNP/indel —— 基本上就是此处观察到的 ~5–6 M 的全部**。这种近乎完全的重叠意味着该背景不是工程位点附近的小段连锁残余,而是一个**大范围、全基因组的 129 型基因组**。要给出确切供体品系(以及纯 129 还是 129×B6 混合)的定论,可按需运行专门的 strain-assignment 分析。
- MGP 扣除后,每样本余下 **~0.57–0.66 百万 private 变异**,在所有染色体上均匀分布(正比于染色体长度)—— 即 MGP 未覆盖的残余背景分歧,而非局部 de-novo 事件。

真正的因果性 de-novo 事件数量级是几十到几百,而非几十万。**要从这个背景中把它们识别出来,需要功能上限定为高影响编码后果(frameshift / stop-gain / splice,经 VEP 或 snpEff),并叠加复现性与 L1L2-vs-L1L2H 差异过滤。** 该功能注释步骤是推荐的下一步分析;各样本 private call set 已随交付提供(`candidates_denovo/*.private.vcf.gz`)。关键在于,这不影响 Study B 的首要结论:copy-number 分析(§6.2)已确立这些基因组结构稳定且为二倍体。

### 6.10 输卵管表型 —— 已测试与未测试的部分

这是 Study B 存在的意义所在,故我们直接陈述其状态,而不是让读者从上面各节自行推断。三种机制可能在工程元件按设计未激活的小鼠中产生年龄依赖的输卵管表型。**只有一种被测试过。**

| 假设(优先级) | 状态 | 我们能说什么 |
| :--- | :--- | :--- |
| **1. iHPV 插入突变** —— 构建体整合进并打断某个基因 | **未测试** | 已确立 transgene 在 L1L2H 存在、在 L1L2 不存在(§6.7)。**整合到了哪里尚未解析**,故是否打断基因未知。需要完整载体图谱以捕获跨结合部读段。 |
| **2. 渗漏 / 体细胞 loxP 重组** —— 年龄依赖的 Cre 非依赖切除,删除 Lats1/2 外显子或去阻遏 E6/E7 | **未测试** | 判读依据是 EGFP "stop" 盒是否完整,而我方 EGFP 标记序列是错误的密码子变体,故该检验从未真正运行(§6.7)。Lats1/2 floxed 外显子完整性同样未评估。 |
| **3. 大尺度基因组不稳定 / aneuploidy** | **已测试 —— 阴性** | 六个组织、两种基因型、所有年龄,copy-number 图谱平坦二倍体(§6.2)。该机制**已排除**,且这个排除是扎实的:Study A 肿瘤充当阳性对照,证明方法在确实存在非整倍体处能检出。 |
| **3b. De-novo 点突变/indel** | **未完成** | 各样本 private 变异集已产出(§6.9),但尚未限定到高影响后果,故不存在候选基因清单。 |

**因此诚实的立场是:我们排除了染色体不稳定这一原因,而尚未测试研究设计所列为最可能的那两个机制。** 本报告的 Study B 结果不应被读作"未发现基因组原因"。

**有一个结构性观察现在就与生物学相关,并且约束了假设 1。** 你报告**两个**品系都有异常,但只有 L1L2H 携带 iHPV 构建体 —— L1L2 根本没有 transgene(§6.7,所有年龄构建体读段均为零)。因此 iHPV 整合**无法解释 L1L2 的表型**。两个品系**共有**的东西才是更值得看的地方,而这样的东西有两个:

- **floxed Lats1/2 等位本身。** 内含子里的 loxP 插入可以降低所在基因的表达;floxed 等位在 Cre 之前并不总是表型沉默的。若 Lats1/2 任一等位是 hypomorphic,则两个品系从出生起就带有降低的 Hippo 通路剂量 —— 这是一条在完全没有 Cre 的情况下、通向缓慢的年龄依赖表型的合理路径。**WGS 无法测试这一点** —— 需要受累组织的 Lats1/2 RNA 或蛋白 —— 但它与此处观察到的一切相符,包括基因组稳定且二倍体。
- **品系背景。** 这些小鼠不是同源纯 C57BL/6J,携带全基因组范围的非-6J(最可能 129 型)背景(§6.9)。若"异常"是相对 C57BL/6 的预期来判断的,那么部分表型可能是品系固有特征,而非工程改造的后果。一只同背景的野生型同窝小鼠就能立刻把两者分开。

### 6.11 体细胞点突变(Study A)—— 尚不能作为可用的突变负荷

五个配对全部跑了 Mutect2 tumor-vs-origin(`somatic/somatic_counts.tsv`):

| 配对(vs RO_origin) | PASS 总数 | SNV | Indel |
| :--- | ---: | ---: | ---: |
| RO_B1TP | 90,680 | 49,965 | 22,613 |
| RO_B2TP | 87,901 | 48,822 | 22,257 |
| RO_tumor1 | 86,549 | 49,886 | 21,772 |
| RO_tumor2 | 79,920 | 47,965 | 18,324 |
| RO_tumor3 | 305,795 | 247,362 | 45,537 |

**列出这些数字只是为了说明它们为何不可用 —— 请不要把它们当作突变负荷引用。** 内部对照证明了这一点:**RO_B1TP 与 RO_B2TP 是亲本细胞在 sgRNA 电穿孔后仅传了几代的产物。** 生物学上它们相对 RO_origin 除切点外几乎不该有任何差异 —— 屈指可数的几个事件。它们却各报出约 88,000–91,000 个。那整条 ~90,000 的地板因此是**假象而非生物学**,原因是本次运行缺少 contamination 估计与 panel-of-normals,导致反复出现的测序/比对错误被当作 somatic call 通过。

在这个地板的背景下读,唯一突出的特征是 **tumor3 高出地板约 3.4 倍**(305,795 vs ~80,000–91,000 基线)—— 方向上与它作为重排最重基因组的地位一致(§6.2、§6.8)。我们把这标为"方向一致",而非定量结果:如此巨大的假象地板无法可靠扣除,且超出的一部分本身可能反映 tumor3 的非整倍体与结构异常劣化了局部比对。

要引用真实负荷,需要 §7.3 所列的过滤(VAF/深度阈值、orientation-bias 过滤、panel-of-normals)。在此期间,Study A 的结论建立在 copy number(§6.2)、编辑验证(§6.3)、身份(§6.4)与 SV(§6.8)之上,均不依赖这些计数。

### 7.1 对模型的启示

**tumor3 成瘤了,并且成为三个瘤中最非整倍体的那个,却不带任何预期的 Brca1/Brca2/Pten 编辑。** 实践后果是具体的:注射细胞群中编辑逃逸的细胞仍能成瘤,因此在未对该具体肿瘤做基因分型之前,不应把观察到的肿瘤表型归因于 Brca/Pten 缺失 —— tumor3 就是明证。

### 7.2 驱动肿瘤的是什么:Trp53 丢失,以及 tumor3 的全基因组灾难

**每个肿瘤都丢失了 Trp53 杂合性;没有一个未成瘤细胞丢失**(§6.5)。tumor1 的纯合区是 *focal* 的 —— Trp53 所在的 69–70 Mb 处 92%,而两侧仅 3 Mb 外为 1% 与 3% —— 这是选择性丢失 Trp53 直接写在数据里。tumor2 经由 chr11 大片段抵达同一终点,tumor3 经由全基因组 LOH。三个独立肿瘤、三种不同机制、一个汇聚的靶点。

这提供了编辑无法解释的那个驱动因素。tumor3 **不带**任何预期的 Brca1/Brca2/Pten 病灶,却是队列中最非整倍体的基因组;答案是它走了另一条路抵达同一处 —— **全基因组 copy-neutral LOH**(§6.6),它让细胞在每一个抑癌基因上同时纯合,Trp53 也在内。tumor1 与 tumor2 把工程化的 Brca1+Pten 敲除与一次 Trp53 second hit 结合起来;tumor3 则免去了工程改造,把 Trp53 —— 以及其它一切 —— 整体丢掉。

一个需要精确陈述的限制:LOH 确立了每个肿瘤只保留**一条** Trp53 单体型、因而是纯合的,但**保留的是哪一条**(工程 null 还是野生型)无法从这些数据读出,因为该工程病灶太小、在 coverage 下不可见(六个样本包括亲本的 Trp53 读段深度都是平的)。保留野生型那条会恢复 p53 功能、不带来优势,故"丢失野生型等位"是压倒性可能的解读。若你们提供该 "−" 等位的构建体设计,我们可以直接在现有数据上把它坐实 —— 无需新测序。

### 7.3 后续分析(除注明外无需客户输入)

1. **iHPV 整合位点 —— Study B 最有价值的待办分析。** 构建体存在已确认,但整合到哪里、是否打断基因尚未解析(§6.10 假设 1)。需要**完整载体图谱**(PMC4662542)以捕获跨结合部读段。注意:它至多只能解释 L1L2H,解释不了 L1L2。
2. **lox-stop-lox 盒完整性**(§6.10 假设 2)—— 检验 E6/E7 是否已因渗漏重组被去阻遏,以及 Lats1/2 floxed 外显子是否完整。需要载体图谱里**真实的 EGFP 序列**;我方标记用了错误的密码子变体,故该检验尚未运行(§6.7)。
3. **受累组织的 Lats1/2 表达** —— 不是 WGS 分析,但它针对的是与数据最相符的那个假设(hypomorphic floxed 等位影响两个品系,§6.10)。需要输卵管的 RNA 或蛋白。
4. **一只同背景的野生型同窝小鼠** —— 可立即把"品系固有特征"与"工程改造的后果"分开(§6.10)。一次 WGS 甚至基因分型即可。
5. **体细胞点突变负荷** —— 引用数字前需要 §6.11 所述的额外过滤。
6. **Study B de-novo 候选挖掘** —— 对已提供的 private call set 作功能高影响注释(VEP/snpEff)+ 复现/基因型差异过滤(§6.9)。推荐的下一步分析。
7. **Study B strain assignment** —— 可选;可确定背景是纯 129 还是 129×B6 混合。
8. **肿瘤保留了哪条 Trp53 单体型** —— 肿瘤在 Trp53 纯合已确立(§6.5);要确认存活的那条是工程 null 而非野生型,需要你方提供该 "−" 等位的构建体设计(§7.2)。这是唯一需要客户输入的项目,且它细化的是解读,不改变任何结论。
