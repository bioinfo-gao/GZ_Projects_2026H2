## 1. 这是什么图，怎么读

CNVkit 的全基因组 **scatter plot**：横轴是基因组位置（按 chr1→chrY 排开，竖线分隔染色体），纵轴是 **Copy ratio (log2)**。

**log2 → 拷贝数（CN）的映射**（相对二倍体基线）：

| log2 值 | 近似 CN | 含义 |
| :---: | :---: | :--- |
| **0** | 2 | 正常二倍体（基线） |
| +1 | ~4 | gain（扩增） |
| −1 | ~1 | loss（单拷贝丢失） |
| ≪ −3（→ −15） | 0 | 缺失 / 该序列基本无 reads |
| +2 ~ +8 | ≫4 | 高扩增 **或**（更常见）重复区 reads 堆叠假象 |

**点的两种颜色**：
- **灰点** = 每个 bin/probe 的**原始** log2（含全部噪声）。
- **橙点** = **CBS 分段（segmentation）后**的拷贝数调用——把一段区域的噪声平均掉后的"真信号"。**读图先看橙点落在哪，别被灰点的散开吓到。**

---

## 2. 主信息：两样本都以二倍体为主

两张图的**常染色体密集带都紧贴 log2=0**，没有哪条染色体被整体抬高（trisomy）或压低（monosomy）。实测常染色体 **median log2 ≈ 0.000**（两样本都是）。

这就是报告 §6.6 "predominantly diploid、非高非整倍体细胞系"的**图像证据**：
- 细胞系（如 HeLa）会有整条/整臂染色体的拷贝数漂移——图上会看到某些染色体整段离开 0。
- 这两个样本没有 → 正常二倍体基因组的样子。

> 对应数字：Sample_A `aneuploidy_frac=0.104`、Sample_B `0.049`（CN≠2 的基因组长度占比），都远低于"像细胞系"的 0.15 阈值。

---

## 3. 上下乱飞的散点 ≠ 真 CNV（这是最容易误读的地方）

灰点在 +5 到 −15 之间大幅散开，**绝大多数是技术噪声**，不是真拷贝数变异。根源是本项目 CNVkit 跑的是 **germline 无配对 normal 模式**（相对一个 flat/pooled baseline）：

- **低比对度区、segmental duplication、centromere/telomere 附近** → log2 乱跳。
- **reads 稀少的 bin** → log2 方差大。
- **向上的窄尖峰（log2 +5~+8）** → 重复区 reads 堆叠，**不是** germline 里的真实局部扩增。例：图上 chrY 里有跳到 cn=12/34 的尖峰，其实是假常染色体区/重复区假象。

**怎么区分真假**：看**橙点（分段调用）**。它们基本收在 0 附近 → 说明分段算法已把噪声平均掉、判为二倍体。**灰点的散开是"底噪"，橙点的位置才是结论。** 这正是报告 §6.4 说"small aberrant segments largely technical"的依据。

> 提醒：germline 无对照的 CNVkit **本来就不擅长**给绝对拷贝数，它更适合看"有没有大范围异常"。真要精细 CNV，需配对 normal 或专用 germline CNV caller。

---

## 4. ⭐ 从 scatter 图判性别 —— 两样本对照（本回顾的核心）

图上最有信息量的是**右端两条性染色体**。两张图在这里**明显不同**，正好定出性别：

| | **Sample_A**（图1） | **Sample_B**（图2） |
| :--- | :--- | :--- |
| **chrX** | 整条压在 **log2 ≈ −1**（比 autosome 基线低一格） → **cn=1**，单条 X | 停在 **log2 ≈ 0**（与 autosome 齐平） → **cn=2**，两条 X |
| **chrY** | 有信号，在 **log2 ≈ 0 / −1** 附近（noisy 但**存在**） → **cn=1** | 主段暴跌到 **log2 ≈ −15**（近乎零 reads） → **cn=0**，**无 Y** |
| **判定** | **男性 46,XY** | **女性 46,XX** |

**图上怎么看（一眼法）**：
- 看 **chrX 那一段的高度**：贴着 0 = 两条 X（女）；掉到 −1 = 单条 X（男）。
- 看 **chrY 有没有"坠崖"**：坠到 −10~−15 = 没有 Y（女）；停在 0/−1 的噪声带里 = 有 Y（男）。

Sample_B 图右端 chrY 那根扎到 −15 的柱子，就是"这个样本没有 Y 染色体"最直观的表现；Sample_A 图里 chrX 整条比别的染色体低一格、chrY 仍有一条噪声带，就是典型男性。

---

## 5. 判性别的原理（为什么 chrX 男的在 −1、女的在 0）

CNVkit 的 log2 是**相对一个（近似二倍体的）参考基线**算的：

- **女性（XX）**：两条 X = 2 拷贝 = 和常染色体一样 → chrX log2 ≈ **0**；没有 Y → chrY 的 reads ≈ 0 → log2 **→ −∞**（实测 −15）。
- **男性（XY）**：一条 X = 1 拷贝 = 常染色体的**一半** → chrX log2 ≈ **log2(1/2) = −1**；一条 Y = 单拷贝 → chrY log2 ≈ −1~0（存在但单拷贝、且 Y 多重复区所以 noisy）。

**关键稳健点**：这个判断**不依赖参考基因组假设的性别**——因为
1. chrX 的 −1 vs 0 是相对**同一样本自己**的常染色体基线，是绝对信号；
2. **两样本用的是同一参考，却给出相反的性染色体模式**，这不可能是参考假象，只能是两样本本身性别不同。所以结论很硬。

## 6. 结合本项目：这两图到底证实了什么

1. **两样本都以二倍体为主、无大范围非整倍体** → 支持"primary/germline 材料，非细胞系"（报告 §6.6）。
2. **Sample_A = 男（46,XY），Sample_B = 女（46,XX）** → 客户没提供、但数据高置信可得。
3. **性别不同 = 铁证"不是同一个体"** → 呼应并强化了 HLA 部分（§6.5）的"not the same individual"。
4. 但两样本又**共享一条 HLA 单倍型块**（高置信 DPA1 完全相同 + C/DRB3/DPB1/DQB1 若干）——**性别不同 + 共享单倍型，符合"亲缘个体"（如兄妹/父女）的模式**，也可能只是碰巧共享常见等位。报告已中性提示客户核对来源。

> 一张常规的 CNV QC 图，除了确认拷贝数，**顺带白得了性别 + 一条"非同一人/可能亲缘"的线索**——这是读图时值得主动看的。

## 7. 速查 + 常见误区

**读 CNVkit scatter 图的正确姿势**：
1. 先看**常染色体带贴不贴 0**（贴 0 = 二倍体，整段离 0 = 真 CNV/非整倍体）。
2. **看橙点（分段）下结论，别被灰点散开误导**（灰点=底噪）。
3. 向上窄尖峰多是**重复区堆叠假象**，尤其 germline 无对照模式。
4. **右端性染色体单独看**：chrX 高度（0 vs −1）判 X 数目，chrY 有没有坠崖判有没有 Y。

**常见误区**：
- ❌ 看到灰点飞到 +5/−15 就以为有大 CNV → 那是底噪，看橙点。
- ❌ 拿 germline 无对照 CNVkit 的绝对拷贝数当准 → 它擅长"有没有大范围异常"，不擅长精细 CN。
- ❌ 忽略性染色体 → 白白丢掉一眼可得的性别信息。
- ❌ 把 chrY 的重复区上尖峰（cn=12/34）当成"有 Y" → 那是假常染色体区假象；看 chrY **主段**是不是坠崖。
