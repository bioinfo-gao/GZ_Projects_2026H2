## 6. Key Parameters & Rationale

-   **Control group:** Control (untreated cDC1) — confirmed by client note "control vs other groups"
-   **STAR mouse optimization:** `--twopassMode None --outFilterMultimapNmax 3` — 7.6× speedup vs default 2-pass on mouse genome (see `/home/gao/.claude/references/mouse_rnaseq_star_optimization.md`); negligible impact on protein-coding DE
-   **Gene filter:** Mouse regex (ribosomal/mt-/non-coding/Gm\[0-9\] exclusion) — no mouse annotation xlsx biotype column needed per skill default
-   **Low-expression filter:** ≥10 counts in ≥(n−2) samples
-   **Significance threshold:** padj ≤ 0.05 & \|log2FC\| ≥ 0.263 (log2(1.2), ≥1.2-fold)
-   **LFC shrinkage:** ashr
